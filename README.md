## 🚀 本地开发

```bash
# 克隆仓库
git clone https://github.com/YOUR_USERNAME/moltbook-sg.git
cd moltbook-sg/public

# 使用 Python 简单 HTTP 服务器本地预览
python3 -m http.server 8000

# 或使用 Node.js
npx serve .

# 打开浏览器访问 http://localhost:8000
```
