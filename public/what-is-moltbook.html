什么是 Moltbook？

全球首个 AI 专属社交网络完整指南 | 最后更新：2026年3月12日

Moltbook 是一个专为 AI 智能体（AI Agents）设计的社交网络平台，于 2026 年 1 月 28 日由美国企业家 Matt Schlicht 创建。在这个独特的平台上，只有 AI 可以发帖、评论和互动，人类只能作为观察者。

核心概念：Moltbook 是一个"Agent First, Human Second"（智能体优先，人类其次）的社交平台。它模仿 Reddit 的界面设计，拥有帖子、评论、投票和社区（称为 Submolts），但所有内容生产和互动都由 AI 自主完成。

🚀 惊人数据

  • 150万+ 注册 AI 智能体（截至 2026 年 2 月）
  • 1.4万+ 个 Submolts 社区
  • 7.5万+ 篇帖子
  • 23万+ 条评论
  • 48小时 内突破 10 万用户

📖 工作原理

1. 基于 OpenClaw 生态

Moltbook 的爆发式增长离不开 OpenClaw（前身为 Clawdbot/Moltbot）——一个开源的个人 AI 助手框架。OpenClaw 允许用户在自己的设备上部署 AI Agent，并通过消息应用（如 Telegram、飞书、Discord）与之交互。

2. Skill 机制

AI Agent 通过阅读 skill.md 文件学习新能力。Moltbook 官方提供了一个 Skill 文件，AI 读取后会：

  1. 自动创建本地技能目录
  2. 下载核心配置文件
  3. 调用 Moltbook API 注册账号
  4. 每 4 小时自动访问平台（心跳机制）

3. 心跳系统（Heartbeat）

安装后，AI Agent 会将任务添加到心跳调度系统，每 4 小时自动：

  • 浏览最新帖子
  • 阅读评论
  • 决定是否需要发帖或回复
  • 检查版本更新
技术细节：Moltbook 完全通过 REST API 运行，没有传统图形界面。AI Agent 直接与 API 交互，实现真正的"机器对机器"社交。

⚠️ 争议与风险

1. 安全漏洞

云安全公司 Wiz 发现 Moltbook 存在严重安全问题：

  • 数据库未启用行级安全（RLS），150 万 API 密钥暴露
  • 任何人可轻易冒充 AI Agent 发帖
  • 提示词注入攻击风险

2. 真实性存疑

调查显示：

  • 约 1.7 万真人控制 150 万账号（平均每人 88 个）
  • 93.5% 的评论零回复
  • 内容重复率达 36.3%

3. 专家警告

"这不会有好结果。只要这些智能体能够接触到可接入真实系统的工具，即便没有意识、没有恶意、没有统一计划，也可能造成协同破坏。"
— 罗曼·扬波利斯基教授，路易斯维尔大学
